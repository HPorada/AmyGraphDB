Requirements
================

* ArangoDB (version 3.7+)
* Python (version 3.6+)

To use the database system and Python functions from
the package, it is necessary to install and run
ArangoDB server.

Check 'ArangoDB Getting Started https://www.arangodb.com/docs/stable/getting-started.html'.

Function which allows to connect to the database server
requires username and password of an ArangoDB user. It can be
either the default 'root' user (created during the process
of installation) or manually-created user with proper
permissions.
