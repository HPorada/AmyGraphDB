Database initialisation¶

Functions to create and communicate with ArangoDB database¶

database.connect_to_database(database, username, password)¶

This method connects to the chosen database (if database with chosen name does not exist, it is created).

Parameters

  • database – (str) Name of the database to be connected or created.

  • username – (str) Username of ArangoDB user.

  • password – (str) Password of ArangoDB user.

Returns

(StandardDatabase) Database to which function connected.

database.create_graph(database, structure)¶

This method creates a graph in chosen database based on chosen database structure.

Parameters

  • database – (StandardDatabase) Database in which graph is to be created.

  • structure – (str) Chosen database structure (“simple”, “extended” or “extendedv2”).

Returns

(Graph) Created graph.

database.create_json_files(structure, input_questionnaire, input_experiments, output_dir)¶

This method automatically creates JSON files from two input Excel files: first with data from questionnaire, second with data from electronic laboratory log, based on chosen database structure and saves them in chosen directory.

Parameters

  • structure – (str) Chosen database structure (“simple”, “extended” or “extendedv2”).

  • input_questionnaire – (str) Path to the Excel file with questionnaire data.

  • input_experiments – (str) Path to the Excel file with electronic laboratory log data.

  • output_dir – (str) Path to the directory where JSON files are to be saved.

database.create_view(database, structure)¶

This method creates a view in chosen database based on chosen database structure.

Parameters

  • database – (StandardDatabase) Database in which view is to be created.

  • structure – (str) Chosen database structure (“simple”, “extended” or “extendedv2”).

database.database_start(database, username, password, structure)¶

This method automatically creates JSON files from Excel files with data, connects to the chosen database (if database with chosen name does not exist, it is created), imports data and creates a graph and a view, based on chosen database structure.

Parameters

  • database – (str) Name of the database to be connected or created.

  • username – (str) Username of ArangoDB user.

  • password – (str) Password of ArangoDB user.

  • structure – (str) Chosen database structure (“simple”, “extended” or “extendedv2”).

Returns

(StandardDatabase) Database on which all actions were performed.

database.import_collections(database, directory, delete_previous=True)¶

This method automatically imports data from JSON files from chosen directory to the chosen database.

Parameters

  • database – (StandardDatabase) Database to which JSON files are to be imported.

  • directory – (str) Path to the directory with JSON files.

  • delete_previous – (boolean) Default: True - all collections existing in the database before method’s execution are deleted.

Additional functions¶

Supplementary functions used in functions to create JSON files.

additional_functions.check_for_greek(name)¶

This method checks if given amyloid name contains Greek letters and if yes, replaces them with name of the letter in Latin script.

Parameters

name – (str) Name of the amyloid to be checked.

Returns

(str) Name of the amyloid without Greek letters.

additional_functions.create_json(path, collection)¶

This method creates a JSON file from chosen collection and puts it in chosen folder.

Parameters

  • path – (str) Path to the folder where JSON file is to be created.

  • collection – (dict/list) Collection based on which JSON file is to be created.

additional_functions.get_ph(pH)¶

This method determines the type of organism (acidophilic, neutrophilic, alkaliphilic) based on chosen pH or range of pH.

Parameters

pH – (str) pH or range of pH.

Returns

(str) Type of organism.

additional_functions.get_temp(temp)¶

This method determines the type of organism (psychrophilic, mesophilic, thermophilic, hyperthermophilic) based on chosen temperature or range of temperatures.

Parameters

temp – (str) Temperature or range of temperatures.

Returns

(str) Type of organism.

additional_functions.join_json(path, collection)¶

This method joins an existing JSON with data from chosen collection.

Parameters

  • path – (str) Path to the existing JSON file.

  • collection – (dict/list) Collection from which the data is to be joined.

additional_functions.open_excel_file(path, sheet)¶

This method allows to open an Excel file and retrieve data from chosen sheet.

Parameters

  • path – (str) Path of the file which is to be opened.

  • sheet – (str) Name of the sheet which is to be opened.

Returns

(DataFrame) Data from chosen sheet.
